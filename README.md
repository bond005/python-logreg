Модуль logreg.py создан в учебных целях для практических занятий по курсу "Методы и алгоритмы компьютерной лингвистики", читаемому студентам Гуманитарного факультета Новосибирского государственного университета.

В модуле logreg.py реализован алгоритм логистической регрессии для решения задачи классификации. Процесс обучения логистической регрессии рассматривается как максимизация логарифма функции правдоподобия градиентным методом.

Модуль logreg.py разработан на языке программирования Python 3.x с использованием Python-библиотеки NumPy. Модуль может быть использован как компонент любой Python-программы, в которой необходимо решать задачу классификации с помощью алгоритма логистической регрессии. Кроме того, данный модуль может быть запущен и как самостоятельная программа: в таком случае будет продемонстрирован процесс обучения распознаванию рукописных цифр из обучающей подвыборки корпуса MNIST, а затем - процесс распознавания рукописных цифр из тестовой подвыборки корпуса MNIST. Чтобы запустить модуль logreg.py как самостоятельную программу, необходимо установить в системе ещё и Python-библиотеку ScikitLearn (средствами этой библиотеки организован доступ к MNIST).
